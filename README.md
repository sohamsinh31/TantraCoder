# Tantra

**Tantra** is a robust, fully-featured AI coding agent that runs completely offline or in the cloud. No ChatGPT. No Copilot. No API keys required. No internet needed (unless you choose to connect). 

It reads your project files, understands the context, writes code, and fixes bugs automatically—powered by any LLM you choose. This is no longer just a lightweight tool; Tantra is a heavy-duty development powerhouse you fully own and control.

## Key Features

* **Local Mode:** Your code never leaves your machine. Completely private execution using local models.
* **Cloud Models & AWS Hosted GPUs:** Scale your capabilities by pointing Tantra at any remote LLM API, including powerful AWS-hosted GPUs for maximum generation speed and complex reasoning.
* **Context-Aware:** Reads existing project files to seamlessly understand your codebase and architecture.
* **Autonomous Bug Fixing:** Detects and fixes bugs automatically without ever leaving your terminal.
* **Zero Dependencies on Proprietary APIs:** Run entirely on open-source or custom models.
